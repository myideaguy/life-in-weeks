# Your Life in Weeks

A memento-mori web tool inspired by Wait But Why's "Your Life in Weeks." Visitors take a short quiz (birthday, gender, a few lifestyle questions) and see a personalized chart of their life broken down into weeks (or months), with how much is already lived, how much each habit (phone, work, sleep, etc.) consumes, and how much truly free time remains.

## Run locally

It's a single static HTML file. Either:

- Double-click the **Life in Weeks** launcher on your Desktop (it starts a small local server and opens the tool in your browser), or
- From this folder, run `python3 -m http.server 8765` and open `http://localhost:8765/`.

## Deploy

This deploys to Vercel as a static site — no build step needed.
